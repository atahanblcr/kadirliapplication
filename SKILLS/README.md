## 🎯 Nasıl Kullanılır?

### Görev Başlamadan Önce:

```
1. Hangi skill gerekli? → Listeden seç
2. SKILL dosyasını oku
3. Best practices'i uygula
```

### Örnek:

```
Görev: "Auth modülü yaz"

Adımlar:
1. backend-nestjs.md oku → Module structure
2. api-security.md oku → JWT, rate limiting
3. testing-strategy.md oku → Test yaz
4. Kodu yaz (skill'lerden öğrendiklerinle)
```

## 💡 Skill Ekleme

Yeni bir skill gerekiyorsa:

```bash
1. SKILLS/ klasörüne yeni .md dosyası ekle
2. Format: [skill-name].md
3. İçerik: Best practices, örnekler, antipatterns
4. README.md'ye ekle
```
