## Performance Tips

1. **Code Splitting**: Use `next/dynamic` for heavy components
2. **Image Optimization**: Use `next/image` for all images
3. **Server Components**: Default to Server Components, use Client only when needed
4. **Caching**: Utilize Next.js data cache and request memoization
5. **Bundle Size**: Monitor with `@next/bundle-analyzer`
